## MODIFIED Requirements

### Requirement: Productos Destacados para Clientes

El sistema SHALL mostrar una selección extendida de productos destacados (7-8 productos) en la interfaz principal o en la interfaz de cliente una vez que la sesión esté iniciada.

#### Scenario: Visualización de productos al iniciar sesión

- **WHEN** el usuario inicia sesión exitosamente
- **THEN** se le presenta una selección de 7-8 productos recomendados con el estilo visual de la tienda
